FGLServer非盈利性服务器公告

发表于2021-11-28|更新于2021-11-28|Minecraft
|字数总计:559|阅读时长:1分钟|阅读量:

与同学合作搭建的非盈利性自由Minecraft Java Edition 1.16.5国际版服务器将于近期重新上线，后续1.18上线后会考虑升级核心版本。此次为服务器第二次对外试开放，搭建目的在于交流学习娱乐，Java版玩家如有意向一起玩请私聊，欢迎各路生存红石建筑PVP等玩家前来玩耍，如有加入意向，请发送邮件至FGLChina@outlook.com，请在邮件中说明您的意向并附上您的正版ID，审核通过后将单独发送客户端和服务器地址并将您的ID加入白名单，网易版玩家免谈。

详细情况

服务器实行强制白名单制，为了提高玩家素质启用了正版验证，盗版玩家和未在白名单中的玩家无法连接，服务端没有加入拓展性mod，因此客户端只能安装辅助性mod，否则尝试连接可能会导致您的客户端崩溃。对服务器建设做出重大贡献的，将考虑给予二级op权限。

服务器默认游戏模式为生存，出生点保护半径为8。修改规则为死亡不掉落和禁止生物破坏方块，防止您的努力功亏一篑，如有需要，可申请导出地图数据自用。

服务器没有任何行为和领地限制，因此是完全自由的，这意味着不会阻止不道德的行为，达成真正的“开放世界”，与此同时，服务器的加入审核机制也会十分严格，尽可能避免恶意玩家和低素质玩家的加入。

FGLServer非盈利性服务器欢迎您的到来。交流学习生活和技术，结识志同道合的玩家是搭建此服务器的根本目的。

注意事项
1.建议使用Java16环境来运行客户端
2.如果恶意增加服务器计算负担，如建造并运行大规模高频电路，将按后台数据调查永久封禁您的IP。
